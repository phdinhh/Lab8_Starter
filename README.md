**How are graceful degradation and service workers related?**
The two concepts are related becase they're both related to ensuring the user continues to be able to use an application even when there are problems that may cause the application to break down. Specifically, graceful degradation is building an application with the maximum technology while still addressing the lower levels. This ensures that even if a few components break, the application isn't entirely broken down. Service workers is similar in that sense, as service workers will store network requests and their responses locally. This ensures that users will still be able to access the application even when internet breaks down through allowing the user to view cached data.
